# Logs

## Player Activity

Retrieve player activity

### Request

**URL** : `/api/v1/logs/player/[lookupKey]/activity`

**Method** : `Character's id.`

**Lookup Key:** : `GET`

**Authorization Required** : `Yes`

**Content Type** : `NA`

**Body**
Empty

---

### Response

**Condition** : NA

**Code** : `200 SUCCESS`

**Example**

```json
{
  "Total": 9,
  "Page": 0,
  "PageSize": 5,
  "Count": 5,
  "Values": [
    {
      "Id": "709e3acb-1c1e-406f-bd0d-a05d538e96b3",
```
