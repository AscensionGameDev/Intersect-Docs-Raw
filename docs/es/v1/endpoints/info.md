# Server Info

## Authorized

Checks token and returns whether or not you're authorized to access protected api endpoints.

### Request

**URL** : `/api/v1/info/authorized`

**Method** : `GET`

**Authorization Required** : `Yes`

**Content Type** : `NA`

**Body**
Empty

---

### Response

**Condition** : NA

**Code** : `200 SUCCESS`

**Example**

```json
{
	"authorized": true
}
```

## Game Name & Port

**Body**
Empty

---

### Response

**Condition** : NA

**Code** : `200 SUCCESS`

**Example**

